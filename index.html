This is a very simple web-page that should help using your phone as a light source for light painting.

How it works:

  • Using the color picker, select colors to add to your palette.
  • Double tap on any color to remove it from the list.
  • Clear the whole palette with the "reset" button.
  • Once you are happy with your selection, click on the "Start" button (you will be redirected to a blank page).
  • Tap the screen once to enter full-screen mode; keep tapping to show the colours of your palette one by one, with black screens in the middle.
  • After all colors have been shown, tapping again will bring you back to the starting page.

Known limitations:

  • Some browsers (especially on phones) do not support full screen mode. In this case, after pressing the "start" buttonm, the very first tap will directly show the first color of the palette.
  • The library that provides the color-picker does not allow to select twice in a row the same color (kinda, the actual issue is a bit more technical). If you need to insert twice in a row the same color, insert it once, then insert another dummy colour, then insert the target colour again and finally get rid of the dummy colour.

Tips:

  • Consider locking the rotation on your phone to avoid the UI moving.
  • If using an iPhone, I suggest using Safari while hiding the toolbar (to do it, at the left of the address bar, click on the "aA" icon, then select "Hide Toolbar") to minimize "UI invasions". If you know that you are going to keep the phone in landscape orientation, do not lock the rotation: the UI invasion in landscape mode is extremely minimal!
